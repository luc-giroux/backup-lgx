Bienvenue sur votre nouveau portail projet! 
De nombreuses fonctionnalités présentes sur le portail Xnet sont disponibles sur ce nouveau portail.
Si vous recherchez un contact, vous pouvez utiliser la nouvelle fonctionnalité présente dans l'onglet "Contacts". Les contacts sont à présent divisés en 5 catégories: Les employés, les entrepreneurs, les salles (où vous pouvez trouver les n° de téléphone des salles de réunion), les bureaux projet, et les contacts divers. Si vous rechercher un numéro radio vous pourrez le trouver ici.
Concernant l'organisation du travail, nous avons toujours les mêmes applications disponibles sous l'onglet Project Control.
Si vous désirez entrer votre feuille de temps, vous pourrez le faire via le menu Timesheet.
Pour approuver les feuilles de temps, l'application est présente sous le menu Approval. Cette application a été légèrement modifiée, vous trouverez de l'aide ici.

Welcome on your new project portal! 
A lot of functionalities available on the XNet portal are now available on this web site.
If you want to search for a contact, you can use the new functionality in tab “Contacts”. The contacts are now divided into 5 categories: Employees, Contractors, Rooms (where you can find the number of meeting rooms), Project Offices (where you have the contact about department) and finally “Various Contacts”. If you are searching for a radio user you can find it here.
About your work organization, we have always the same applications available on the tab Project Control.
If you want enter your timesheet, you will find the application in Timesheet menu.
To approve the timesheets, the application is available in the "Approval" menu. This application has been slightly modified, you will find help here.
